# Personal Portfolio - Engy Hassan

A modern, responsive personal portfolio website showcasing frontend development skills with a focus on user experience and performance.

## 🚀 Features

- **Responsive Design**: Fully responsive across all devices
- **Dark/Light Mode**: Toggle between dark and light themes
- **Multiple Themes**: Choose from various color schemes
- **Font Options**: Switch between different Arabic fonts (Tajawal, Cairo, Alexandria)
- **Portfolio Gallery**: Filterable portfolio items
- **Testimonials Carousel**: Interactive testimonials section
- **Smooth Animations**: CSS animations and transitions
- **Accessibility**: ARIA labels, skip links, and keyboard navigation
- **SEO Optimized**: Meta tags and structured data
- **Performance Focused**: Optimized fonts, images, and code

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Tailwind CSS framework with custom styles
- **JavaScript**: Vanilla JS for interactivity
- **Tailwind CSS**: Utility-first CSS framework
- **Font Awesome**: Icons library
- **Google Fonts**: Custom Arabic fonts (Alexandria, Tajawal, Cairo)

## 🚀 Getting Started

### Prerequisites

- A modern web browser
- No build tools required (static site)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/personal-portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd personal-portfolio
   ```

3. Open `index.html` in your browser or serve it with a local server.

### Development

The project uses Tailwind CSS. To modify styles:

1. Edit the custom CSS in `assets/css/style.css`
2. Or modify the Tailwind classes directly in `index.html`

## 🎨 Customization

### Color Themes

The site includes multiple color themes. To add new themes:

1. Add new color variables in the CSS custom properties
2. Update the theme buttons in the HTML
3. Add corresponding JavaScript logic

### Fonts

Currently supports three Arabic fonts:
- Tajawal (default)
- Cairo
- Alexandria

### Content

Update the following sections in `index.html`:
- Hero section
- About section
- Portfolio items
- Experience timeline
- Testimonials
- Contact information

## 📱 Sections

1. **Hero**: Introduction and navigation
2. **About**: Personal information and skills
3. **Portfolio**: Project showcase with filtering
4. **Experience**: Professional timeline
5. **Testimonials**: Client/peer recommendations
6. **Contact**: Contact form and social links
